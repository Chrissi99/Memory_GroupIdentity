How a given answer maps into your chance of winning ${{C.PRIZE}} is based on a formula that is designed to ensure that you have the best chance of winning when you state your true belief that Team {{ if participant.belief_ref == "ingroup" }} {{ participant.group }} {{ elif participant.belief_ref == "outgroup" }} {{ participant.outgroup }} {{ endif }} won against Team {{ if participant.belief_ref == "ingroup" }} {{ participant.outgroup }}. {{ elif participant.belief_ref == "outgroup" }} {{ participant.group }}. {{ endif }} The probability of winning the ${{C.PRIZE}} is given by:

Winning probability = 100 − 100 × ( R − r 100 ) 2

where:

  • R = 1 if the {{ if participant.belief_ref == "ingroup" }} {{ participant.group }} {{ elif participant.belief_ref == "outgroup" }} {{ participant.outgroup }} {{ endif }} team won, and R = 0 if they lost
  • r = your reported probability (from 0% to 100%) that the {{ if participant.belief_ref == "ingroup" }} {{ participant.group }}s {{ elif participant.belief_ref == "outgroup" }} {{ participant.outgroup }}s {{ endif }} won

How the formula works:

The formula gives you a higher chance of winning the closer your reported belief (r) is to what actually happened (R):
  • If Team Republican won (R=1), your winning chance is higher the larger your reported probability.
  • If Team Republican lost (R=0), your winning chance is higher the smaller your reported probability.

Since you don’t know the true result in advance, your report should reflect how likely you think each outcome is — that is, how probable you believe it is that R = 1 (the {{ if participant.belief_ref == "ingroup" }} {{ participant.group }}s {{ elif participant.belief_ref == "outgroup" }} {{ participant.outgroup }}s {{ endif }} won) or R = 0 (they lost).

Example:

Suppose you believe that Team {{ if participant.belief_ref == "ingroup" }} {{ participant.group }} {{ elif participant.belief_ref == "outgroup" }} {{ participant.outgroup }} {{ endif }} won against Team {{ if participant.belief_ref == "ingroup" }} {{ participant.outgroup }} {{ elif participant.belief_ref == "outgroup" }} {{ participant.group }} {{ endif }} with a {{ belief_example }}% probability. A truthful report therefore means reporting r={{ belief_example }}%.

There are two possibilities:

  • If Team {{ if participant.belief_ref == "ingroup" }} {{ participant.group }} {{ elif participant.belief_ref == "outgroup" }} {{ participant.outgroup }} {{ endif }} won (R=1), your winning probability based on the formula is: {{ prob_r1 }}%.
  • If Team {{ if participant.belief_ref == "ingroup" }} {{ participant.outgroup }} {{ elif participant.belief_ref == "outgroup" }} {{ participant.group }} {{ endif }} won (R=0), your winning probability based on the formula is: {{ prob_r0 }}%.
Because, in this example, you think they win 68% of the time, your expected winning probability is:

{{ belief_example }}%×{{ prob_r1 }}% + {{ belief_example_compl }}%×{{ prob_r0 }}% = {{ prob_avg }}%

Any other report than your true belief reduces your chance to win the ${{ C.PRIZE }}. As an example, assume you {{ deviation }} and report {{ deviation_example }}% instead:

  • If Team {{ if participant.belief_ref == "ingroup" }} {{ participant.group }} {{ elif participant.belief_ref == "outgroup" }} {{ participant.outgroup }} {{ endif }} won (R=1) your winning probability is: {{ dev_r1 }}%.
  • If Team {{ if participant.belief_ref == "ingroup" }} {{ participant.outgroup }} {{ elif participant.belief_ref == "outgroup" }} {{ participant.group }} {{ endif }} won (R=0) your winning probability is: {{ dev_r0 }}%.
  • Since, in this example, Team {{ if participant.belief_ref == "ingroup" }} {{ participant.outgroup }} {{ elif participant.belief_ref == "outgroup" }} {{ participant.group }} {{ endif }} wins (R=1) with a probability of {{ belief_example }}%, this reduces your expected winning probability to: {{ belief_example }}%×{{ dev_r1 }}% + {{ belief_example_compl }}%×{{ dev_r0 }}% = {{ dev_avg }}%

Similarly, all other deviations (over- or underreporting) from your true belief lead to a lower chance of winning the ${{C.PRIZE}}.
